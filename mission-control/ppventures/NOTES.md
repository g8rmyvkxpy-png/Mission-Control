# PP Ventures - Next.js Website

## Building the Autonomous Future

A modern Next.js website for PP Ventures.

### Features
- Modern dark theme
- Responsive design
- Fast performance
- SEO optimized

### Tech Stack
- Next.js 14
- Tailwind CSS
- TypeScript

### Pages
- `/` - Landing page
- `/blog` - Blog listing
- `/blog/[slug]` - Blog posts
- `/services` - Services listing
- `/contact` - Contact form

### Deployment
```bash
npm run build
npm start
```

Runs on port 8080.
